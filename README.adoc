IntelliRoq provides comprehensive IDE support for link:https://iamroq.com[Roq] static site generator projects in IntelliJ IDEA. Build, manage, and preview Roq-powered websites directly from your IDE with intelligent project management, plugin installation, and integrated development workflows.

*Key Features:*

* Automatic Roq project detection and initialization
* One-click creation of data files, layouts, and partials
* Interactive Roq plugin management (10 official plugins supported)
* Integrated Quarkus dev mode with live reload
* Production build generation
* Dedicated tool window for project overview and quick actions

Streamline your Roq development workflow with native IDE integration.

== Features

=== Project Management

* *Automatic Project Detection* - Detects Roq projects by analyzing directory structure and build files
* *Project Initialization* - Create complete Roq project structure with one click
* *File Creation Wizards* - Generate data files (YAML/JSON), layouts, and partials with templates

=== Plugin Management

* *Plugin Detection* - Automatically discovers installed Roq plugins from Maven/Gradle dependencies
* *Interactive Installation* - Add plugins through a selection dialog with descriptions
* *Supported Plugins* (10 official):
** `markdown` - CommonMark content processing
** `tagging` - Dynamic tag collection pages
** `aliases` - URL redirections
** `asciidoc` / `asciidoc-jruby` - AsciiDoc markup support
** `qrcode` - QR code embedding
** `series` - Group related posts
** `sitemap` - Auto-generate sitemap.xml
** `lunr` - Client-side search indexing
** `diagram` - Render diagrams via Kroki.io

=== Development Workflow

* *Quarkus Dev Mode* - Custom run configuration for development mode with live reload
* *Configurable Port* - Set custom port for development server (default: 8080)
* *Console Integration* - View development server output in IDE run window
* *Build System Support* - Automatic Maven/Gradle detection with wrapper support

=== Production Builds

* *Static Site Generation* - One-click production builds from IDE
* *Progress Tracking* - Monitor build progress in console view
* *Build Output* - Generated files in `target/roq/` (Maven) or `build/roq/` (Gradle)

=== Tool Window

* *Project Overview* - View Roq version and directory structure status
* *Quick Actions* - Direct access to common operations (create files, build site)
* *Plugin List* - See installed plugins with versions at a glance
* *Context-Aware* - Interface adapts for Roq and non-Roq projects

== Getting Started

=== Initialize a New Roq Project

. Create a new Java/Maven/Gradle project in IntelliJ IDEA
. Go to *Tools > Roq > Initialize Roq Project Structure*
. Add Roq dependencies to your `pom.xml` or `build.gradle`
. Start creating content!

=== Set Up Development Mode

. Open *Run > Edit Configurations*
. Click *+* and select *Roq Dev Mode*
. Configure port (default: 8080) and environment variables
. Click *Run* to start the development server with live reload

=== Add Roq Plugins

. Go to *Tools > Roq > Add Roq Plugin...*
. Select plugins from the list (e.g., "Sitemap", "Tagging")
. Click *OK* - build file is updated automatically
. Refresh Maven/Gradle dependencies
. Plugins are ready to use!

=== Build for Production

. Go to *Tools > Roq > Build Static Site*
. Monitor build progress in the console
. Find static files in `target/roq/` or `build/roq/`
. Deploy to your hosting provider
